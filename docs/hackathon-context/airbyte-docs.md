## Page Description

Airbyte describes itself as an open source data integration, activation, and agentic data platform. The docs homepage says it helps consolidate data from hundreds of sources into data warehouses, data lakes, and databases, then move data into operational tools such as CRMs, marketing platforms, and support systems.

## Top-Level Docs Entry Points

- Platform: create connections, build custom connectors, and start syncing data
- Connectors: catalog of 600+ sources and destinations
- Release notes: updates for Self-Managed and Cloud
- AI agents: documentation for equipping AI agents to explore and work with data
- Developers: API, Terraform provider, and programmatic integration
- Community and support

## Homepage Notes
