## Page Summary

The welcome page presents Bland as a voice agent platform for inbound and outbound calling. The page copy emphasizes simplicity, security, low latency, and support for both technical and non-technical builders.

## Core Claims On The Page

- Bland says it is fully self hosted, with data remaining secure end-to-end.
- The product is positioned around very low latency so calls sound more human.
- The platform is presented as usable for any agent use case, including by non-technical teams.
- Support is routed through Discord, with live support and daily office hours.
- Discord URL: https://discord.gg/QvxDz8zcKe

## What You Can Do With Bland

- Send calls: dispatch AI phone calls to customers, leads, and operational workflows. https://app.bland.ai/dashboard/send-call
- Set up inbound: create inbound phone numbers for support and related workflows. https://app.bland.ai/dashboard/phone-numbers
- Live API calls during call: connect external APIs and take live actions during calls. https://docs.bland.ai/tutorials/webhooks
- Develop AI Agent personas: create and manage different AI personalities for different use cases. https://docs.bland.ai/tutorials/personas
- Send off batches of calls: send large call batches at once. https://docs.bland.ai/tutorials/batch-calls
- Embed your agents into your website: add AI voice and chat agents to web applications. https://app.bland.ai/dashboard/web-widget
